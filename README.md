I'm **flingo777** — a frontend developer who spends about as much time theming a window manager as writing components. React is home base, but the real hobby is making the whole development environment feel like one deliberately designed system: editor, terminal, compositor, all pulling the same palette.

When I'm not at a keyboard, I'm usually a few episodes behind on something, mid-chapter in a manga, or reinstalling a Linux distro I didn't need to reinstall.
